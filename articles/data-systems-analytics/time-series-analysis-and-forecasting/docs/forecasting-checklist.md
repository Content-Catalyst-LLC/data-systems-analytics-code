# Forecasting checklist

- [ ] Series has a regular or documented time index.
- [ ] Missing periods are documented.
- [ ] Trend, seasonality, cycles, and remainder are inspected.
- [ ] Autocorrelation and lag structure are reviewed.
- [ ] Stationarity or transformation needs are assessed.
- [ ] Forecast horizon is defined.
- [ ] Forecast model family is documented.
- [ ] Backtesting preserves temporal order.
- [ ] Rolling-origin evaluation is used where appropriate.
- [ ] Error is reported by horizon.
- [ ] Prediction intervals are published with point forecasts.
- [ ] Structural breaks and regime changes are monitored.
