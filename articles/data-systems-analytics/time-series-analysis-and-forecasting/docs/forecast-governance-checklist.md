# Forecast governance checklist

- [ ] Forecast owner is assigned.
- [ ] Forecast release status is tracked.
- [ ] Horizon and interval assumptions are documented.
- [ ] Forecast accuracy is monitored over time.
- [ ] Structural-break review is scheduled.
- [ ] Forecasts are connected to decision risk.
- [ ] High-consequence forecasts include uncertainty intervals.
- [ ] Forecast revisions and model changes are versioned.
