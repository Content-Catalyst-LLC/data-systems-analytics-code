# Causal question checklist

- [ ] Intervention is clearly specified.
- [ ] Comparison condition is clearly specified.
- [ ] Outcome is measured after treatment.
- [ ] Unit of analysis is clear.
- [ ] Time horizon is clear.
- [ ] Estimand is named: ATE, ATT, LATE, CATE, ITT, or TOT.
- [ ] Assignment mechanism is documented.
- [ ] Identification assumptions are stated.
- [ ] Threats to validity are reviewed.
- [ ] Robustness and sensitivity checks are planned.
