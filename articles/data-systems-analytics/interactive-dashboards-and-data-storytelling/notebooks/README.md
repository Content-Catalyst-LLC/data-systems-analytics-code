# Notebook guidance

Use notebooks to inspect dashboard integrity, not to hide official dashboard logic.

Recommended pattern:

1. Run `python/dashboard_storytelling_scorecard.py`.
2. Load `outputs/dashboard_storytelling_manifest_python.json`.
3. Inspect dashboard-level integrity scores, KPI context, filter burden, linked view design, story coherence, annotation quality, accessibility, governance, and evidence traceability.
4. Compare monitoring dashboards, exploratory dashboards, guided stories, operational dashboards, and legacy dashboards.
5. Keep exploratory dashboard analysis separate from approved publication artifacts.
