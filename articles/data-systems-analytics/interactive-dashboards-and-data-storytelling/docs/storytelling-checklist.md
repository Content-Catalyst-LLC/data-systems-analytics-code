# Data storytelling checklist

- [ ] Story has a clear analytical question.
- [ ] Story points follow a meaningful sequence.
- [ ] Each story point has a claim and support.
- [ ] Narrative emphasis is proportional to evidence.
- [ ] Uncertainty and caveats appear near the claim.
- [ ] Counter-patterns or limitations are not hidden.
- [ ] Annotations clarify rather than decorate.
- [ ] Interaction supports the story rather than derailing it.
- [ ] The final point distinguishes finding, implication, and action.
- [ ] Evidence links and review status are preserved.
