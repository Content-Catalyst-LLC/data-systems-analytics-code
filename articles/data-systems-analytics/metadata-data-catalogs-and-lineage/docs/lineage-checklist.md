# Lineage checklist

- [ ] Upstream datasets are identified.
- [ ] Downstream assets are identified.
- [ ] Transformations are linked to jobs or code.
- [ ] Column-level lineage exists for critical metrics and sensitive fields.
- [ ] Manual steps are documented.
- [ ] Provenance records include entities, activities, and agents.
- [ ] Runtime lineage is connected to observability.
- [ ] Lineage is usable for impact analysis.
- [ ] Semantic changes are documented.
- [ ] Lineage confidence or granularity is visible to users.
