# Catalog review checklist

- [ ] Asset is discoverable.
- [ ] Description is complete and current.
- [ ] Owner and steward are visible.
- [ ] Business definition is linked to glossary.
- [ ] Technical schema is current.
- [ ] Policy tags are visible.
- [ ] Quality signals are surfaced.
- [ ] Lineage is visible.
- [ ] Downstream consumers are listed.
- [ ] Trust label is justified by evidence.
