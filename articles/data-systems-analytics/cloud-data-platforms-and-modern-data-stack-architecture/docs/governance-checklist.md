# Governance checklist

A modern data stack should be governed by design.

- [ ] Sensitive sources are classified.
- [ ] Access policies are tied to identity and role.
- [ ] Data catalogs include owner, description, freshness, and sensitivity.
- [ ] Lineage connects downstream assets to upstream sources.
- [ ] Semantic metrics are certified before broad use.
- [ ] Raw zones have retention and access limits.
- [ ] Curated zones include quality tests.
- [ ] Warehouse marts include documented grains and definitions.
- [ ] AI/ML feature assets include provenance and drift monitoring.
- [ ] Audit logs are retained for access and change events.
