# Responsible-use review checklist

- [ ] Use case is clearly stated.
- [ ] Purpose is compatible with original collection or approved reuse.
- [ ] Sensitive attributes are identified.
- [ ] Data minimization has been considered.
- [ ] Fairness or disparate impact risk has been reviewed where relevant.
- [ ] Privacy and security controls are documented.
- [ ] Retention and deletion expectations are clear.
- [ ] Human accountability is assigned.
- [ ] Appeal, correction, or contestability path exists where appropriate.
- [ ] Decision and conditions are recorded.
