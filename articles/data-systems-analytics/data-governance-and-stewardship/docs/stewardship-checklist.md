# Stewardship checklist

- [ ] Data owner is identified.
- [ ] Data steward is identified.
- [ ] Decision rights are explicit.
- [ ] Business definitions are current.
- [ ] Critical data elements are identified.
- [ ] Quality thresholds are defined.
- [ ] Issue triage path exists.
- [ ] Classification is visible.
- [ ] Access review process is documented.
- [ ] Lifecycle and retention controls are documented.
- [ ] Responsible-use review is required for high-impact reuse.
- [ ] Escalation path is active.
