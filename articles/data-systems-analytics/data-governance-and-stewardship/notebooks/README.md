# Notebook guidance

Use notebooks to inspect governance maturity, stewardship queues, policy coverage, access decisions, lifecycle controls, and responsible-use risks.

Recommended pattern:

1. Run `python/governance_stewardship_scorecard.py`.
2. Load `outputs/governance_stewardship_manifest_python.json`.
3. Inspect asset-level governance maturity, gaps, quality issues, access reviews, lifecycle controls, and responsible-use risks.
4. Compare certified, reviewed, uncertified, restricted, active, and deprecated assets.
5. Keep exploratory governance analysis separate from approved policies and decision-rights records.
