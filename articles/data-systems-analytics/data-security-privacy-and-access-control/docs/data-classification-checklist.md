# Data classification checklist

- [ ] Asset owner is assigned.
- [ ] Personal data flag is set.
- [ ] Direct identifiers are tagged.
- [ ] Regulated or contractual obligations are documented.
- [ ] Sensitivity tier is assigned.
- [ ] Retention period is documented.
- [ ] Allowed purposes are documented.
- [ ] Downstream flows inherit or transform classification.
- [ ] Public aggregates are reviewed for inference risk.
- [ ] Secrets and credentials are separated from analytical data.
