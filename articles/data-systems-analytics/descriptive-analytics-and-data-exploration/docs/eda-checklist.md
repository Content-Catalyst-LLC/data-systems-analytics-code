# EDA checklist

- [ ] Data dictionary or variable profile exists.
- [ ] Row counts and unique identifiers are checked.
- [ ] Missingness is summarized by variable and subgroup.
- [ ] Numeric distributions are inspected with robust summaries.
- [ ] Categorical domains and sparse levels are reviewed.
- [ ] Outliers are investigated before removal.
- [ ] Aggregates are compared against subgroup summaries.
- [ ] Bivariate and multivariate relationships are explored.
- [ ] Time structure is inspected where relevant.
- [ ] Questions generated by EDA are preserved.
