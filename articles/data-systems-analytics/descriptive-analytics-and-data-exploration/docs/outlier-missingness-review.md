# Outlier and missingness review

Outliers and missing values should become questions before they become deletions.

Ask:

- Is the value possible?
- Is it a coding, unit, timestamp, or entry issue?
- Is it concentrated in one system, group, or period?
- Is it a rare but real event?
- Does it change the mean more than the median?
- Does missingness follow a pattern?
- Will downstream models treat missingness as random when it is not?
