# Notebook guidance

Use notebooks to inspect exploratory evidence, not to hide the official profile logic.

Recommended pattern:

1. Run `python/descriptive_eda_scorecard.py`.
2. Load `outputs/descriptive_eda_manifest_python.json`.
3. Inspect numeric profiles, categorical profiles, subgroup summaries, missingness, bivariate relationships, aggregation risk, and EDA questions.
4. Compare the overall summary against segment-region summaries.
5. Preserve questions generated by exploration before moving to inference, prediction, or causal analysis.
