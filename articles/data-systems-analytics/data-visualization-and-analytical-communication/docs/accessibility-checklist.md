# Visualization accessibility checklist

- [ ] Text and labels are readable.
- [ ] Color contrast is sufficient.
- [ ] Color is not the only indicator.
- [ ] Patterns, labels, or symbols reinforce color.
- [ ] Tooltips are not the only source of essential meaning.
- [ ] Captions explain the chart in plain language.
- [ ] Exported images include alt text.
- [ ] Screen-reader or text alternative is available for critical visuals.
- [ ] Dense graphics have summary interpretation nearby.
- [ ] The visual can be understood in grayscale.
