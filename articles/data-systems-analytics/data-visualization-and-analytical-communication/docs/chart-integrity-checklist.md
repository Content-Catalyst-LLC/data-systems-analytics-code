# Chart integrity checklist

- [ ] Chart type matches analytical task.
- [ ] Axis baseline and scale are appropriate.
- [ ] Categories are ordered meaningfully.
- [ ] Units, denominator, period, and population are visible.
- [ ] Color is not the only signal.
- [ ] Important uncertainty is visible near the claim.
- [ ] Annotation clarifies rather than decorates.
- [ ] Caption explains the interpretive point.
- [ ] Source asset and method reference are linked.
- [ ] Output is versioned if consequential.
