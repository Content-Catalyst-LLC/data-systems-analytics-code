# Notebook guidance

Use notebooks to inspect visualization integrity, not to hide official visual evidence logic.

Recommended pattern:

1. Run `python/visualization_integrity_scorecard.py`.
2. Load `outputs/visualization_integrity_manifest_python.json`.
3. Inspect chart fit, encoding quality, uncertainty, annotation, accessibility, evidence traceability, audience fit, review score, and output control.
4. Compare executive, technical, policy, operational, legacy, and model-evaluation visuals.
5. Keep exploratory chart design separate from approved publication artifacts.
