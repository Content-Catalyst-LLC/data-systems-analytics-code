# Predictive monitoring checklist

- [ ] Prediction distribution is tracked.
- [ ] Input drift is tracked.
- [ ] Performance drift is tracked when labels arrive.
- [ ] Calibration drift is reviewed.
- [ ] Threshold volume and review capacity are monitored.
- [ ] Subgroup performance is monitored where relevant.
- [ ] Metric deterioration triggers escalation.
- [ ] Model version, data version, and monitoring window are linked.
