# Interoperability governance checklist

- [ ] Technical exchange is monitored.
- [ ] Schemas and message contracts are versioned.
- [ ] Semantic mappings have owners.
- [ ] Controlled vocabularies are governed.
- [ ] Entity crosswalks are reviewed.
- [ ] Lineage events are emitted.
- [ ] Security and privacy boundaries are enforced.
- [ ] Payloads are minimized by purpose.
- [ ] Standards adoption is documented.
- [ ] Local distinctions are preserved when unification would distort meaning.
