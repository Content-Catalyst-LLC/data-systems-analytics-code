# Survivorship checklist

- [ ] Attribute authority is defined.
- [ ] Source priority is documented.
- [ ] Recency rules are explicit.
- [ ] Conflicts trigger review where needed.
- [ ] Legal identifiers block unsafe merges when conflicting.
- [ ] Contextual views are preserved where one golden value would distort meaning.
- [ ] Effective dates are maintained.
- [ ] Overwrites preserve history.
- [ ] Stewardship exceptions are auditable.
- [ ] Downstream users understand which view they are consuming.
