# Root cause and stewardship notes

Data defects often reveal process defects.

Typical causes include:

- optional fields that should be required
- poorly validated input forms
- incompatible source-system definitions
- weak master data management
- ungoverned manual workarounds
- undocumented transformations
- unstable identifiers
- missing data ownership

Quality management improves the process that produces data, not only the records that contain defects.
