# Notebook guidance

Use notebooks to inspect data product readiness, not to hide certified logic.

Recommended pattern:

1. Run `python/product_scorecard.py`.
2. Load `outputs/product_platform_manifest_python.json`.
3. Inspect product readiness, usage, and quality summaries.
4. Compare certified products against beta, deprecated, and uncertified products.
5. Keep exploratory analysis separate from certified semantic logic.
