# Monitoring checklist

- [ ] Monitoring metric windows are defined.
- [ ] Acceptable performance limits are documented.
- [ ] Drift indicator is tracked.
- [ ] Performance is reviewed across groups and cohorts.
- [ ] Calibration is reassessed after deployment.
- [ ] Threshold remains appropriate for operational capacity.
- [ ] Escalation conditions are explicit.
- [ ] Model owner and steward are assigned.
- [ ] Retraining, rollback, or retirement path exists.
