# Threshold policy guidance

A classification threshold is a decision policy, not only a model parameter.

Document:

- positive class meaning
- threshold value
- false-positive cost
- false-negative cost
- precision expectation
- recall expectation
- review capacity
- affected population
- escalation path
- approved owner
- review cadence

When false negatives are more costly, a lower threshold may be justified. When false positives create material harm or review burden, a higher threshold may be necessary.
