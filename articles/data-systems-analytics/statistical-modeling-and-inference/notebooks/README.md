# Notebook guidance

Use notebooks to inspect statistical evidence, not to hide official inference logic.

Recommended pattern:

1. Run `python/statistical_inference_scorecard.py`.
2. Load `outputs/statistical_inference_manifest_python.json`.
3. Inspect group intervals, mean differences, regression estimates, diagnostics, robustness flags, and readiness scores.
4. Compare two-sample estimation, regression, adjusted regression, and weak p-value threshold workflows.
5. Keep exploratory analysis separate from final inferential claims.
