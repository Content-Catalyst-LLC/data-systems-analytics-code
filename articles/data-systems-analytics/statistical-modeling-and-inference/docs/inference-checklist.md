# Statistical inference checklist

- [ ] Estimand is defined.
- [ ] Population or process of interest is stated.
- [ ] Sampling or data-generation process is documented.
- [ ] Point estimate is reported.
- [ ] Standard error or uncertainty measure is reported.
- [ ] Confidence interval or credible interval is reported where appropriate.
- [ ] Hypothesis test is interpreted cautiously.
- [ ] Effect size is discussed separately from p-value.
- [ ] Model assumptions are reviewed.
- [ ] Diagnostics and robustness checks are preserved.
- [ ] Practical meaning is interpreted in context.
