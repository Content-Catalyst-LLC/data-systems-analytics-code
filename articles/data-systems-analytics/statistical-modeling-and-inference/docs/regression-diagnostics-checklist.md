# Regression diagnostics checklist

- [ ] Residual patterns inspected.
- [ ] Nonlinearity assessed.
- [ ] Heteroskedasticity assessed.
- [ ] Influential observations reviewed.
- [ ] Multicollinearity reviewed where multiple predictors are used.
- [ ] Outliers distinguished from measurement errors.
- [ ] Subgroup instability reviewed.
- [ ] Robustness checks documented.
